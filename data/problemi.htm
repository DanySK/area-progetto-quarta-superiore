Requisiti di sistema:
Processore Intel Pentium o Celeron o AMD K6 o superiore (consigliati almeno 400MHz)
32MB di RAM (consigliati 64MB)
Sistema operativo Microsoft Windows 98 o successivo (non sono stati effettuati test su sistemi operativi Unix)
Scheda grafica e monitor in grado di supportare la risoluzione di 1024x768 pixels con colore a 16bit per pixel o superiore

L'animazione in alto non viene visualizzata
Scaricare l'ultima versione di Macromedia Flash Player (freeware) a questo indirizzo

Non si riesce a consultare la Legge Regionale 10/02
Procurarsi una copia di Microsoft Word. Il file è stato testato solo sulla versione XP ma dovrebbe funzionare correttamente anche sulla versione 2000.

Impossibile aprire "Nozioni di illuminotecnica"
Installare Adobe Reader 6.0 in versione freeware, semplicemente clickando qui. Non è necessaria la connessione ad Internet, il file è presente sul CD.

Impossibile aprire il file contenente la copertina del lavoro
E' sufficiente installare il programma apposito (CD Box Labeler Pro) in versione freeware semplicemente clickando qui. Una volta installato, è sufficiente aprire questo file e stampare (utilizzare l'opzione front/backside).
